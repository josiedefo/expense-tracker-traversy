# Expense Tracker

A simple expense tracking application to help you manage your personal finances.

## Features

- Add and categorize expenses
- View expense history
- Track spending by category
- Calculate totals and summaries

## Installation

```bash
git clone https://github.com/yourusername/expense-tracker-traversy.git
cd expense-tracker-traversy
npm install
```

## How to Run the Application

1. Clone the repository:
    ```bash
    git clone <repository-url>
    cd expense-tracker
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Start the development server:
    ```bash
    npm start
    ```

4. Open your browser and navigate to `http://localhost:3000` (or the port shown in your terminal).

The application will automatically reload when you make changes to the source files.

## Getting Started from Scratch

1. Create a new directory for your project:
    ```bash
    mkdir expense-tracker
    cd expense-tracker
    ```

2. Initialize a new npm project:
    ```bash
    npm init -y
    ```

3. Create the basic file structure:
    ```bash
    mkdir src css js
    touch index.html src/app.js css/style.css
    ```

4. Set up a basic HTML template in `index.html`:
    ```html
    <!DOCTYPE html>
    <html lang="en">
    <head>
         <meta charset="UTF-8">
         <meta name="viewport" content="width=device-width, initial-scale=1.0">
         <title>Expense Tracker</title>
         <link rel="stylesheet" href="css/style.css">
    </head>
    <body>
         <div id="app"></div>
         <script src="src/app.js"></script>
    </body>
    </html>
    ```

5. Open `index.html` in your browser or use a local development server
